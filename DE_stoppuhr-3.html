Deine Stoppuhr - Bonus

Hier sind ein paar Ideen, wie du deine Stoppuhr noch verbessern könntest. Wenn du selber noch Ideen hast, kannst du gerne mithilfe des Coding Club Teams versuchen, sie umzusetzen.

Vergesse nicht, nach deinen Änderungen dein Programm neu herunterzuladen !

Immer zwei Ziffern anzeigen

Im Moment zeigt die Uhr nur dann zwei Ziffern anm wenn es nötig ist. Wie du programmieren kannst, dass die uhr statt “3:9” lieber “03:09” anzeigt, erklären wir hier.

Wir schreibe dafür zuerst eine Funktion, die zum Beispiel “9” durch “09” ersetzt.

Schreibe unter der Funktion stop() eine Funktion formatUhr(i), die kontrolliert ob i kleiner als 10 ist und falls ja eine Null davor setzt.
Zum Schluss spuckt die Funktion diese modifizierte Version von i zurück. Schreibe dafür als letzte Zeile return i;

Tipp (Klicke auf den Pfeil falls du einen Tipp Brauchst) Kontrollieren ob i kleiner als 10 ist kann man mit
if (i < 10) {
}
Eine Null vor i setzten kann man mit der Zeile i = "0" + i;

Nun wollen wir immer bevor wir die Uhr anzeigen das Format aktualisieren.

Finde dafür alle Stellen in deinem Code an denen document.getElementById("uhr").innerHTML = min +":" + sec; vorkommt und ersetze diese Zeile durch die drei Zeilen minString = formatUhr(min);
secString = formatUhr(sec);
document.getElementById("uhr").innerHTML = minString +":" + secString;

Eine Schöne Schrift

Bis jetzt wird deine Uhr nur in sehr simpler Schrift und ganz klein angezeigt. In diesem Abschnitt geben wir dir die Möglichkeit, deine Uhr etwas schöner zu gestalten.

Füge dafür direkt ganz oben unter der zweiten Zeile deines Programms ( <html> ) folgenden Codeblock ein:
<head>
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>Mein Stoppuhr</title>
<style>
#uhr {
font-size: 2em;
margin: 20px;
font-family: monospace;

}
button {
font-size: 1em;
padding: 10px 20px;
cursor: pointer;
font-family: system-ui;
}
</style>
</head>
Was passiert wenn du die verschiedenen Zahlen die vor "px" oder "em" vorkommen veränderst? Kannst du herausfinden welche Zahl was verändert? Schaffst du es dein Uhr zentriert auf dem Bildschirm anzuzeigen?
Zusätzlich kannst du die Schriftart verändern, indem du das Wort hinter font-family: durch einen der Folgenden Liste ersetzt:

Arial

Times

Trebuchet MS

Georgia

Garamond

comic sans MS

Verdana

Courier

Courier New

Lucida

Helvetica

Brush Script MT

Impact

Tahoma

Werde kreativ!

Natürlich gibt es noch ganz viele andere Möglichkeiten, um die Uhr noch weiter zu verschönern und verbessern. Versuche auch gerne deine eigenen Ideen dafür aus.
