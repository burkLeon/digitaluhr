Nun ist es an der Zeit!

Mit dem Wissen, das du bis hier angesammelt hast, kannst du jetzt eine Stoppuhr für den Hasen zu Programmieren. Und es wird Zeit ! Der Hase bekommt schon langsam angst, zu spät beim Ball der Königin aufzutauchen.
Im Codeblock unten wollen wir die Stoppuhr programmieren.

Das Display

Beginne damit, das Display für unsere Stoppuhr einzubauen. Erstelle dafür in der Zeile direkt unter
<!-- Ab hier kommt der angezeigte Inhalt -->
einen div-Container mit id "uhr" zu erstellen, die den Text "00:00" anzeigt. Schaffst du es zusätzlich diesen Container zentriert auf der Webseite anzuzeigen?

Tipp (Klicke auf den Pfeil falls du einen Tipp Brauchst) Um den Text anzuzeigen solltest du folgende Codezeile an der erwähnten Stelle hinzugefügt haben:
<div id="uhr"> 00:00 </div>
Um diese in der Mitte der Webseite anzuzeigen, kannst du diese Zeile in einem center Container einbetten.

Die Variablen

Diese angezeigte Uhr wollen wir mit Variablen zum Leben bringen, damit sie auch gut die Zeit stoppen kann.

Definiere unter der Zeile // Ab hier kommt der Code für die Uhr // eine Variable sec , die als Anfangswert die Zahl 0 hat. Diese Variable wird die Sekunden zählen. Füge auch eine Variable min mit Wert 0 hinzu, die die Minuten zählt.

Tipp (Klicke auf den Pfeil falls du einen Tipp Brauchst) Erinnere dich daran, dass Variablen mit
var [Name] = [Wert];
definiert, wo [Name] der Name der Variable ist und [Wert] ihr Anfangswert.

Tick, Tick, Tick…

Das Funktionieren der Stoppuhr ist im Code zwischen den Klammern { und } bei der Zeile function stoppuhr() geregelt.

Füge zwischen den Klammern { } drei Zeilen in folgender Reihenfolge ein.
1. Eine Zeile, um den Wert von der Variable sec um 1 zu erhöhen;
2. Eine Zeile, die den angezeigten Wert aktualisiert, und zwar:
document.getElementById("uhr").innerHTML = min +":" + sec;
3. Eine Zeile, um eine Sekunde zu warten, bevor das Programm erneut abgerufen wird:
setTimeout(stoppuhr, 1000);

Tipp (Klicke auf den Pfeil falls du einen Tipp Brauchst) Den Wert von sec kannst du mit folgendem Code um 1 erhöhen:
sec = sec + 1;
oder
++sec;

Nun läuft unsere Stoppuhr von ganz alleine !! Das Aussehen ist vielleicht noch nicht ganz das erwünschte, aber darum kümmern wir uns später.

Sekunden zu Minuten

60 Sekunden sind eine Minute. Das weiss jedes Kind, nur unser Programm noch nicht. Wenn du die Stoppuhr laufen lässt, zählt sie nach 60 einfach weiter mit 61, 62, 63, und so weiter. Das wollen wir nun ändern, und dem Programm sagen, dass 60 Sekunden eine Minute ist.

Füge unter der Zeile, die die Sekunden um 1 erhöht einen neuen Codeblock ein. Dieser sollte kontrollieren, ob 60 Sekunden vergangen sind, und falls ja die Minutenanzahl min um eins erhöhen und sec wieder auf null zurücksetzen. Benutze dafür den Codeblock
if (sec == 60){

}
dessen Inhalt in den { } Klammern nur dann ausgeführt wird, wenn sec gleich 60 ist.

Tipp (Klicke auf den Pfeil falls du einen Tipp Brauchst) Die Minuten kann man auf der gleichen Art und Weise um 1 erhöhen wie die Sekunden. Um sec auf Null zu setzen benutzen wir sec = 0; . Frage eine Assistentin oder einen Assistenten falls du mehr Hilfe brauchst.
