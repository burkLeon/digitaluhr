Variablen

Deine Uhrzeit muss in deinem Programm gespeichert werden. Dafür braucht es Variablen. Das sind wie Schubladen, um Infos zu speichern. Wir haben dir wieder ein paar Aufgaben vorbereitet, die du direkt in der Box hier lösen kannst.
Zwischen den grünen Linien im oberen Abschnitt gibt es einen Teil für den Text-Inhalt und im unteren einen Teil für dein Code. Achte gut darauf, was wohin kommt!

Den Namen merken

Eine Variable kann zum Beispiel einen Namen speichern.

Schreibe im Inhalt einen ersten div-Container mit dem Text "Mein Name ist:" und einen zweiten div-Container mit dem Text "???".
Den zweiten Container wollen später noch verändern. Darum geben wir ihm eine id indem wir im Start-Tag ein id="name" hinzufügen:
<div id="name"> ??? </div>
Jetzt können wir im Code-Teil unten eine Variable erstellen. Schreibe im Code-Abschnitt die Zeile:
var meinName = Sophie;
Die drei Fragezeichen können wir dank der id jetzt mit der Variable ersetzen. Schreibe im Code-Abschnitt die Zeile: document.getElementById("name").innerHTML = meinName;
Jetzt steht auf der Webseite rechts "Sophie" statt "???" ! Versuche im Code "Sophie" durch deinen Namen zu ersetzen. Was passiert?

Dank der id können wir mit unserem Programm auf den Inhalt unserer Webseite zugreifen und ihn verändern. Das wird für unsere Uhr sehr wichtig! Dafür brauchen wir auch variablen, wie hier “name”.
Variablen werden immer gleich definiert:

Variablen werden immer so definiert: var [NameDerVariable] = [Startwert];

Die Uhrzeit wissen


Jetzt können wir die allererste Version unserer Digitaluhr programmieren. Und es wird Zeit! Der Hase verliert schon die Hoffnung, also los geht's!
Du kannst die vier Zeilen die du davor programmiert hast löschen oder einfach darunter weitermachen.
Lasse dich vom Code für den Namen inspirieren um die Uhr zu programmieren. Es ist recht ähnlich.

1. Erstelle im Inhalt einen div-Container mit dem Text "00:00" und id "uhr".
2. Erstelle im Code eine Variable "stunde" und eine Variable "minute" und setze sie auf die aktuelle Uhrzeit.
3. Schreibe im Code folgende Zeilen um die angezeigte Uhrzeit mit deinen Variablen zu füllen:
document.getElementById("uhr").innerHTML = stunde + ":" + minute;

Tipp (Klicke auf den Pfeil falls du einen Tipp Brauchst) 1. Benutze folgende Zeile im Inhalt Teil:
<div id="uhr"> 00:00 </div>
2.Schreibe im Code Teil. Dein Code könnte so aussehen
var stunde = 11;
var minute = 36;
