Wenn alles gut gelaufen ist, solltest du jetzt ein Display haben, dass die Sekunden zählt. Ganz zufrieden ist der Hase aber noch nicht, denn die Zeit läuft von alleine los und wird nur auf Null zurückgesetzt, wenn man die Seite neu lädt. "Was ist denn das für eine Stoppuhr, bei der man die Zeit nicht Stoppen kann?" wundert es sich.
Das wollen wir nun ändern, indem wir Knöpfe für stop, start und reset hinzufügen.
Deinen bisherigen Code für die Uhr findest du auf dieser Seite etwas weiter unten wieder

Die Zeit stoppen

Füge unter der Zeile <div id="uhr"> 00:00 </div> einen Knopf mit onclick "stop()" und Text "stop" mithilfe folgender Codezeile:
<button onclick="stop()"> stop </button>

Dieser code fügt unter unserer Stoppuhr einen Knopf “stop” hinzu, der die Funktion “stop()” aufruft, wenn er angeklickt wird. Vorerst passiert aber noch nichts, wenn du auf den Knopf klickst, denn wir haben die Funktion “stop()” dem Computer noch nicht gegeben.

Füge unter der Zeile var min = 0; eine Variable hinzu mit Namen "lauft" und Anfangswert false .

Tipp (Klicke auf den Pfeil falls du einen Tipp Brauchst) "lauft" ist genauso wie die Zahl Null ein gültiger anfangswert, du kannst also die gleiche Methode wie sonst benutzen: var [Name] = [Wert];

Diese Variable soll speichern, ob die Zeit unserer Stoppuhr gerade läuft (dann ist lauft “true”, oder wahr) oder nicht (dann ist lauft “false”, oder falsch).

Füge unter der Funktion stoppuhr() eine Funktion hinzu mit Namen stop(), die lauft auf "false" setzt.
Die erste Zeile dieser Funktion ist wie die, der Funktion stoppuhr(), aber zwischen den Klammern { } kommt hier der Befehl um lauft auf "false" zu setzen.

Tipp (Klicke auf den Pfeil falls du einen Tipp Brauchst) Die Funktion stop sollte folgendes Format haben:
function stop(){
[hier kommt der Befehl um lauft auf "false" zu setzen]
}
Achte darauf, dass deine Funktion stop() wirklich ausserhalb der { } Klammern der Funktion stoppuhr() ist.

Setze den ganzen Inhalt zwischen den { } Klammern der Funktion stoppuhr() in einem Block der nur dann ausgeführt wird, wenn lauft true ist.
Du kannst dich dafür an der Aufgabe Sekunden zu Minuten orientieren und folgenden Code benutzen:
if (lauft == true){
}

Tipp (Klicke auf den Pfeil falls du einen Tipp Brauchst) Die Funktion stoppuhr() sollte jetzt so aussehen:
function stoppuhr(){

if(lauft==true){


sec = sec+1;


...


}


}

Probiere dein Programm aus. Wenn alles gut gelaufen ist sollte jetzt die Zeit stoppen, sobald du auf den Stop-Knopf drückst.

Die Zeit starten

Nun haben wir es geschafft, die Zeit zu stoppen. Aber wie bringen wir sie wieder zum laufen?
Dafür brauchen wir ganz einfach einen start-Knopf mit einer start() Funktion, die die Zeit wieder zum laufen bringt.

Füge direkt über der Zeile <button onclick="stop()"> stop </button> einen Knopf mit onclick "start()" und Text "start".

Tipp (Klicke auf den Pfeil falls du einen Tipp Brauchst) Inspiriere dich an der Zeile <button onclick="stop()"> stop </button> und ersetze einfach überall "stop" durch "start".

Damit dieser Knopf auch etwas bewirkt, müssen wir wie beim stop-Knopf die Funktion dazu programmieren.

Definiere direkt unter der Funktion stop() eine Funktion start(), die
1. kontrolliert ob lauft den Wert false hat, und falls ja
2. lauft auf true setzt und
3. die stoppuhr Funktion startet mit der Zeile stoppuhr();

Tipp (Klicke auf den Pfeil falls du einen Tipp Brauchst) Erinnere dich daran, dass der Inhalt des Blockes if (lauft==false){

...


} nur dann ausgeführt wird, wenn lauft den Wert false hat.

Die Zeit zurücksetzen

Wenn du willst, kannst du noch einen zusätzlichen Knopf hinzufügen, der die Zeit wieder auf Null stellt, bevor du auf der nächsten Seite weitermachst.

Füge direkt unter der Zeile <button onclick="stop()"> stop </button> einen Knopf mit onclick "reset()" und Text "reset".

Tipp (Klicke auf den Pfeil falls du einen Tipp Brauchst) Inspiriere dich an der Zeile <button onclick="stop()"> stop </button> und ersetze einfach überall "stop" durch "reset".

Definiere direkt unter der Funktion start() eine Funktion reset(), die
1. beide Variablen min und sec auf Null stellt,
2. lauft auf false setzt, und
3. die angezeigte Zeit mit folgender Zeile aktualisiert : document.getElementById("uhr").innerHTML = min +":" + sec; .
