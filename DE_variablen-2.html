Variablen ändern

Die Uhrzeit mit Variablen ist gut. Aber es ist besser, wenn sie sich auch auf Befehl aktualisiert, damit sie pünktlich bleibt. Wie man Variablen verändern kann erkunden wir jetzt. Deine Uhr von der vorherigen Seite solltest du in der Box unten wiederfinden, frage sonst gerne um Hilfe !

Um Inhalt zu verändern, müssen wir die Variablen verändern. Das geht am besten mit einer Funktion. Wir definieren jetzt zusammen unsere erste Funktion.

Achte gut auf Gross- und Kleinschreibung und Sonderzeichen im Code, sie spielen eine wichtige Rolle !

Wir schreiben eine Funktion mit Namen erhoeheStunde, die die Variable "stunde" um eins erhöht und dann das Anzeigefeld mit id "uhr" aktualisiert. Der Code dafür ist wie folgt: function erhoeheStunde(){
stunde = stunde + 1;
document.getElementById("uhr").innerHTML = stunde + ":" + minute;
}
Kopiere diesen Code unter deinem existierenden Code.

Um die Funktion aktivieren zu können, werden wir einen Knopf verwenden.

Im Inhalt-Teil, füge unter der Zeile für die Uhr einen button-Container hinzu mit Text "+ stunde".
Füge dann im Start-Tag Text hinzu wie folgt (ähnlich zur id):
<button onclick="erhoeheStunde()">

Tipp (Klicke auf den Pfeil falls du einen Tipp Brauchst) Deine Zeile Code sollte wie folgt aussehen :
<button onclick="erhoeheStunde()"> + stunde </button>

Jedes mal, wenn du jetzt rechts auf den Knopf klickst, wird die Stundenzahl um eins erhöht!

Erstelle mit der gleichen Methode einen Knopf, der die Minuten um eins erhöht. Das heisst:
1. Erstelle eine Funktion mit Namen erhoeheMinute die "minute" um eins erhöht und dann das Anzeigefeld mit id "uhr" aktualisiert.
2. Erstelle im Inhalt einen Knopf mit Namen "+ minute" und onclick "erhoeheMinute()".

Tipp (Klicke auf den Pfeil falls du einen Tipp Brauchst) 1. Kopiere die Struktur der Funktion erhoeheStunde() aber ändere stunde zu minute wo nötig. Achte darauf, dass die neue Funktion unterhalb des vorherigen Codes anfängt. Frage um Hilfe falls du mehr Hilfe brauchst.
2. Definiere den Knopf direkt unter den vorherigen. Der Code für diesen Knopf ist <button onclick="erhoeheMinute()"> + minute </button>
